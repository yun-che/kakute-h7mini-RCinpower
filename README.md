## 配件组装
- 固定机臂、白色打印件至碳板
- 焊接
1. 电调-电机：电调左上、左下、右上、右下各有3个焊点，分别对应四个电机3*4条线，电机三条线不必按某个顺序，因为每两条线调换后代表电机转向反向，后面在飞控配置可以调节（tips：__先将电调安在机臂上绕好线再焊，否则焊好后无法再将电机放在机臂上__）
2. 电调-电源供电线
3. 飞控-接收机
- 安装飞塔（电调在下，飞控在上）
